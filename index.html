Recycler

Créer du CSS générique

Il est préférable d'écrire des classes avec peu de propriétés et facilement réutilisable. Cela évite la duplication de code et d'écrire directement dans le fichier html.

CSS

    .clear {
        clear:both;
    }

    .floatRight {
        float:right;
    }

    section.block {
        width: 600px;
        margin: 0 auto;
    }

Toutes les éléments possédant la classe block sera centré au milieu de l'écran. Seul le menu aura un autre positionement.

HTML

    <header class="block">HEADER</header>

    <nav class="floatRight">MENU<nav>

    <section class="block diaporama">DIAPORAMA</section>

    <section class="block">CONTENU</section>

    <footer class="block">FOOTER</footer>

Éviter de surcharger les classes génériques

    .clear {
        clear:both;
        margin-top:20px;
    }

Il est préférable d'ajouter le margin-top:20px; à une autre balise, ou de créer une autre classe.

    .clear {
        clear:both;
    }

    .clear.marginTop {
        margin-top:20px;
    }

Recycler des classes

Le page possède un header. Un menu a le droit d'en posséder un aussi, ainsi qu'un article. Rien ne nous empêche d'utiliser la même classe header.

On évite d'avoir une trop grande variété de classes avec des noms étranges et on arrive à cibler beaucoup plus facilement le contenu.

HTML

    <nav>
        <div class="header"></div>
        <ul class="content">
            <li>Menu Item #1</li>
            <li>Menu Item #2</li>
            <li>Menu Item #3</li>
        </ul>
        <div class="footer"></div>
    </nav>

    <article>
        <div class="header"></div>
        <div class="content"></div>
        <div class="footer"></div>
    </article>

CSS

    nav .header  { /* MON CSS */ }
    nav .content { /* MON CSS */ }
    nav .footer  { /* MON CSS */ }

    article .header  { /* MON CSS */ }
    article .content { /* MON CSS */ }
    article .footer  { /* MON CSS */ }

Astuces

Privilégier float:left/right; à display:inline-block;

HTML

    <ul>
        <li>Item #1</li>
        <li>Item #2</li>
        <li>Item #3</li>
    <ul>

CSS

    ul li 
    {
        display:inline-block;
    }

Ce choix permet d'éviter les mauvaises surprises des marges magiques de inline-block. La petite astuce pour pallier ce petit soucis :

HTML

    <ul><li>Item #1</li><li>Item #2</li><li>Item #3</li><ul>

Mhm...pas très lisible. Dans ce cas, vous utilisez float:left; ou float:right; sans oublier un petit <div class="clear"></div> après le <ul>

HTML

    <ul>
        <li>Item #1</li>
        <li>Item #2</li>
        <li>Item #3</li>
    <ul>
    <div class="clear"></div>

CSS

    ul li 
    {
        float:left;
    }
